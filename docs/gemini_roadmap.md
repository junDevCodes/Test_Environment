## 1. 현재까지 진행 상황 (As-Is)

### 1.1. 프로젝트 초기 설정
- **프로젝트 구조 분석**: `frontend`, `backend`, `test_set` 등 초기 디렉토리 구조를 분석했습니다.
- **요구사항 반영**: 사용자 피드백을 반영하여 `test.db`를 `prob_db` 디렉토리로 이동하고, `test_set`의 하위 구조(`AI/theory_set`, `AI/practice_set`)를 확정했습니다.

### 1.2. 문제 생성 자동화 스크립트 개발
- `db_generator.py` 스크립트를 생성하여 문제 데이터베이스(`AI_prob.db`)를 자동으로 생성하고 관리하는 기반을 마련했습니다.
- 스크립트 실행 시, 기존 데이터를 모두 삭제하고 새로 문제를 추가하여 항상 최신 상태를 유지하도록 구성했습니다.

### 1.3. 1차 객관식 문제 생성 완료 (총 50문제)
- **[이론] 객관식 (40개)**: `theory_set`의 `정리.ipynb` 파일 내용을 기반으로 생성했습니다.
- **[실습] 객관식 (10개)**: `practice_set`의 정답 Notebook을 기반으로 "How-to" 형식의 문제를 생성했으며, **EDA, 선형 회귀, RAG** 주제를 다루었습니다.
