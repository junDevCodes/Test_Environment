## 2. 앞으로의 계획 (To-Be)

### 2.1. 아키텍처 마이그레이션 (진행 중)
- 현재 프로젝트를 스크립트 기반에서 완전한 웹 서비스 형태로 전환하기 위한 아키텍처 마이그레이션을 진행하고 있습니다.
- **Backend**: Python 기반의 기존 스크립트를 **FastAPI** 프레임워크를 사용한 API 서버로 전환합니다.
- **Frontend**: **React(TypeScript)와 Vite**를 사용하여 사용자에게 퀴즈 기능을 제공하는 SPA(Single Page Application)를 구축합니다.
- **Data Pipeline**: 데이터 생성 워크플로우를 다음과 같이 정의합니다.
    1. **Gemini (AI):** `test_set` 디렉토리의 원본 자료(.ipynb 등)를 기반으로 `data/` 디렉토리에 `[과목]_questions_[종류].json` 형식의 문제 파일 생성.
    2. **DB Generator:** `db_builder/db_generator.py`가 이 JSON 파일들을 읽어 데이터베이스(`.db`)를 생성.
    3. **Validator:** `db_builder/validator.py`를 통해 생성된 데이터베이스의 무결성 및 정확성 검증.
